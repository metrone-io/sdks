Metrone Analytics - Vanilla JS Example

This example demonstrates how to use Metrone Analytics with vanilla JavaScript.

Installation

npm install @metrone-io/sdk

Usage

import Metrone from '@metrone-io/sdk' const analytics = new Metrone({ apiKey: 'metrone_live_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx', endpoint: 'https://api.metrone.io/v1/events' }) // Track page view analytics.pageview() // Track custom event analytics.track('button_click', { button_id: 'example-button' })

React Integration

For React applications, use the separate React package:

npm install @metrone-io/react import { MetroneProvider, useMetrone } from '@metrone-io/react' function App() { return ( <MetroneProvider config={{ apiKey: 'metrone_live_xxx' }}> <MyApp /> </MetroneProvider> ) } function MyComponent() { const { trackEvent } = useMetrone() return ( <button onClick={() => trackEvent('click')}> Track Click </button> ) }
